Franklin is a ChatGPT backed IRC bot

Setup

  1. Install Irssi and the perl interpreter from your system's package manager.

  2. Get it on GitHub

  3. You should configure franklin.pl to autoload on irssi start, and optionally configure your channel autojoins.

  4. Create an API key, then put it in a file called api.key in the scripts working directory.

  5. You’ll need to set up a webserver with a directory under it’s root called said (usually something like /var/www/html/said/), that is writable by the franklin.pl script. Franklin should set the umask correctly for the files automatically.

  6. Edit any variables in franklin.pl you need to ($localdir), such as the domain name of your server where the long-text is hosted.

  7. Finally, test it by calling Franklin: hey buddy, send me a test message so I know that you are working.

Debugging

… Now test the bot by calling it with “Franklin: print me a test message” in channel (do this from a different nick, not Franklin’s nick!)
