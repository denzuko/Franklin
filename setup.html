Franklin is a ChatGPT backed IRC bot

Setup

  1. Install Irssi and the perl interpreter from your system's package manager.

  2. Get it on GitHub

  3. You should configure franklin.pl to autoload on irssi start, and optionally configure your channel autojoins.

  4. Create an API key, then run /set franklin_api_key [api key].

  5. You'll need to create a block.lst file, and point franklin towards it with /set franklin_block_file [filename].

  6. You’ll need to set up a webserver with a directory under it’s root called said (usually something like /var/www/html/said/), that is writable by the franklin.pl script. Franklin should set the umask correctly for the files automatically.

  7. Edit any variables in franklin.pl you need to, but most settings can be changed with the /set franklin... command.

  8. Finally, test it by calling Franklin: hey buddy, send me a test message so I know that you are working.

Debugging

… Test the bot by calling it with “Franklin: print me a test message” in channel (do this from a different nick, not Franklin’s nick!)
You may have errors that franklin can't read from or write to something, check your permissions! (I set ownership to franklin:staff).
If text is to long, try using /set franklin_hard_limit to decrease the maximum number of characters on the line.
After edits you can just do /script load franklin to reload the bot without restarting irssi.
